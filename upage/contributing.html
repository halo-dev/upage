贡献指南

感谢您对 UPage 项目的关注！我们非常欢迎各种形式的贡献，无论是功能开发、bug 修复、文档改进还是使用反馈。本指南将帮助您了解如何参与 UPage 的开发和贡献。

目录​

  • 行为准则 - 参与 UPage 项目的行为准则
  • 贡献方式 - 如何为 UPage 项目做出贡献
  • 开发环境设置 - 如何设置本地开发环境
  • 代码规范 - 代码风格和提交规范
  • 工作流程 - 分支策略、PR流程和版本发布

社区​

加入 UPage 社区，与其他贡献者交流：

  • GitHub Discussions
  • GitHub Issues

许可证​

UPage 采用 基于 GPLv3 的补充协议许可证。通过贡献代码，您同意您的贡献将在相同的许可证下发布。
